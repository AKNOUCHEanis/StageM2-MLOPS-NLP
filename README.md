# StageM2-MLOPS-NLP

Le stage est co-encadré par le Laboratoire Informatique de Paris 6 LIP6 et ZEENEA. Il consiste à utiliser des
approches de traitement automatique du langage naturel avec des modèles de machine learning et deep learning pour
faciliter et/ou automatiser la documentation des patrimoines de données des clients de Zeenea :

* Mise en place d'un pipeline de traitement des données issues de BigQuery
* Création d'un moteur de recherche sémantique (basé sur BERT) pour détecter des liens de similarité entre les
objets d'un catalogue de données (basé sur les documentations textuelles des objets)
* Dockerisation et déploiement sur AWS

Outils : Python, Pytorch, Pandas, BigQuery, SQL, PySpark, MLSPARK, Docker, AWS ( S3, EC2 ), NLP, MLOPS
